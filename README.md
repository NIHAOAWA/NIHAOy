## 安装与部署

### 先决条件

- 前端语言编辑器（推荐SublimeTest)
- 一台电脑
- 域名（如您想做成网站）
- 您的GitHub 账号

### 部署

1. 克隆本仓库（GitHub上）
2. 在编辑器里安装源代码
3. 如您想做成网站，请配置服务器
4. 配置您的域名
5. 部署完成后，请注明本仓库链接以及开发人员
